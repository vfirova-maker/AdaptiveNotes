# Лабораторна робота №10: Адаптивний UI для телефонів/планшетів + доступність + локалізація (uk/en)

Цей проєкт демонструє реалізацію адаптивного застосунку для ведення нотаток **«Adaptive Notes»** на платформі Android з використанням **Jetpack Compose** та **Material 3**.

---

##  1. Мета роботи
* Створення адаптивних макетів для різних розмірів екрана (телефон, планшет).
* Використання breakpoints (Compact, Expanded) через обчислення `WindowSizeClass`.
* Забезпечення доступності: перевірка контрасту, touch-мішені (Touch Targets від 48dp), семантика.
* Реалізація локалізації застосунку на українську та англійську мови.

---

##  2. Реалізація адаптивності та структури UI
Застосунок автоматично підлаштовує інтерфейс під ширину екрана:
1. **Режим телефону (< 600dp):** одноекранна навігація (екран списку змінюється екраном деталей).
2. **Режим планшета (≥ 600dp):** двоколонковий макет (Master-Detail layout) — список відображається ліворуч, а деталі нотатки праворуч.

---

##  3. Доступність (Accessibility)
* **Touch-мішені:** Елементи мають інтерактивну мішень не менше 48dp (`minimumInteractiveComponentSize()`).
* **Семантика:** Додано `contentDescription` для зчитування назв нотаток екранними дикторами.
* **Контраст:** Кольори тексту контролюються через `MaterialTheme.colorScheme.onSurface`.

---

##  4. Локалізація (Localization)
Впроваджено підтримку двох мовних пакетів через ресурси:
* **English:** `res/values/strings.xml`
* **Ukrainian:** `res/values-uk/strings.xml`

Інтерфейс автоматично перекладається відповідно до мови системи пристрою.
